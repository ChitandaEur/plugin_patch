二、安装fcitx5输入法（可跳过，我是方便解释）
sudo pacman -S fcitx5 fcitx5-qt fcitx5-gtk fcitx5-chinese-addons fcitx5-configtool
fcitx5 &
kate /etc/environment
下面三行复制粘贴进去，保存时弹出要求输入密码，我是镜像系统所以不用输入

export GTK_IM_MODULE=fcitx5
export QT_IM_MODULE=fcitx5
export XMODIFIERS=@im=fcitx5

这一步要开启魔法，猫猫安装方法，打开plugin_patch.sh，输入c就可以安装
git clone https://github.com/thep0y/fcitx5-themes.git
cd fcitx5-themes
cp spring ~/.local/share/fcitx5/themes -r
kate ~/.config/fcitx5/conf/classicui.conf
复制下面粘贴到弹出的classicui.conf中

# 垂直候选列表
Vertical Candidate List=False
# 按屏幕 DPI 使用
PerScreenDPI=True
# Font (设置成你喜欢的字体)
Font="Smartisan Compact CNS 13"
# 主题(这里要改成你想要使用的主题名，主题名就在下面)
Theme=spring
#spring 春天
#summer 夏天
#autumn 秋天
#winter 冬天
#green 绿色
#transparent-green 透明绿色
#macOS-light macOS-轻量
#macOS-dark macOS-黑暗模式

四、deb转pkg.tar.zst
完成以后，随便找个deb来试一下
debtap xxxxx.deb
第一个地方填包名，第二个直接回车
在Warning: These dependencies (depend =   fields) could not be translated into Arch Linux packages names:
libatspi2.0-0
这一步，可能会有无法处理的依赖（大部分情况都有），在百度搜索相应的archlinux对应的依赖，替换。（没有的就直接删了那一行）
sudo pacman -U xxxxx.pkg.tar.zst
无法处理，继续搜索或者直接删除，（这里看了下以前怎么处理的，直接删）
可选，直接不选
一般安装完的软件都可以在菜单找到
下面演示todesk，
gtk前面百度网盘就有，直接删没问题
sudo pacman -S libappindicator-gtk2
sudo pacman -S libappindicator-gtk3
每个软件遇到的依赖问题不同，具体问题具体解决
如果提前知道todesk需要libappindicator-gtk3，
我们可以在debtap xxxxx.deb这一步加上它
这样自己打包出的zst安装包别人也可以使用了，更加的方便好用
查看软件所需依赖的命令
ldd 可执行文件位置
如百度网盘是ldd /opt/baidunetdisk/baidunetdisk
